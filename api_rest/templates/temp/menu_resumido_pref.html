    Sessão RH - Recursos Humanos

    O setor de Recursos Humanos (RH) é responsável por gerir os colaboradores de uma empresa, desde a admissão até a demissão. O RH é uma peça-chave no desenvolvimento do capital humano da empresa, pois utiliza softwares de gestão e análise de dados para alinhar os talentos com os objetivos e cultura da organização. Pode ser encontrado a remuneração salarial e os salários dos servidores
